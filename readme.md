# Personal Finance - Streamlit Starter

Minimal personal finance app built with Streamlit + Python + SQLite.

Features:
- Email/password auth (local)
- Accounts (checking, credit, cash, investment)
- Transactions (create, edit, list)
- CSV import (simple mapping)
- Net worth summary & export JSON

Quick start (local):
1. Create a virtualenv:
   ```bash
   python -m venv .venv
   source .venv/bin/activate   # macOS / Linux
   .venv\Scripts\activate      # Windows
